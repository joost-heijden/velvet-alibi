# Mascot Directions

Generated with built-in `$imagegen` mode. Final project assets were copied from `$CODEX_HOME/generated_images/...` into `public/assets/` and optimized for web delivery.

## Selected Mascot

**Vera Lens** is the main mascot: a friendly brass magnifying-glass assistant with a burgundy velvet scarf and bookish case-file energy.

Why selected:

- Strong round silhouette for icon and small UI use
- Detective-coded without copying classic detective characters
- Cuddly and warm, but still adult and premium
- Works as app icon, hint helper, loading/helper visual and share-card character

## Prompt Set Summary

All prompts used the same constraints: no blood, no weapons, no existing detective IP references, no readable text, no watermark, not pink-dominated.

- Mascot concepts: four polished 3D editorial mascot directions for a premium cozy-crime PWA, each with warm paper/brass/oxblood/sage styling.
- App icon: simplified Vera Lens head-and-scarf silhouette, high clarity at small size.
- Assistant image: full-body Vera Lens beside case files, with negative space for UI use.
- Share card: Vera Lens with solved grid, case files and warm desk lighting, leaving negative space for app-rendered share text.
- Small UI illustrations: closed case-file tray, open hint notebook, solved-case seal.
